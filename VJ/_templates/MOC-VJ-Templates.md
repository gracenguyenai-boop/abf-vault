## Cách Dùng

1. Chọn đúng template theo loại note cần tạo
2. Copy vào folder tương ứng của kênh: `VJ/VJ-[tên]/VJ-01-Atomic/[loại]/`
3. Điền YAML frontmatter: thay `branch` = tên kênh, điền `name`, `topics`
4. Feed Loop riêng → dùng `_template-insight` → lưu tại `VJ-01-Atomic/insight/`
